# r04

This repository contains research documentation and supporting files for the paper titled "Digital Transformation and AI Application in Early Childhood Education in Vietnam: Awareness, Readiness, and Barriers".

## Repository structure

- `metadata/`
  - `Metadata.md` — structured metadata and summary information extracted from the source documents.
- `documents/`
  - Supporting research files, including the conference paper, presentation documents, cover and back pages, and data spreadsheet.

## Purpose

The repository is organized to keep metadata and source materials separated for easier navigation and review. Use `metadata/Metadata.md` first for the extracted paper details, then review the original files in `documents/`.

## Files

- `metadata/Metadata.md` — research paper metadata and bibliographic details.
- `documents/Thu moi viet bai HTKH 31.01_OCR.pdf` — scanned or OCR source document.
- `documents/back.pdf` — back page of the paper or cover material.
- `documents/cover.pdf` — cover page of the paper.
- `documents/db.xlsx` — data spreadsheet associated with the research.
- `documents/pp.docx` — presentation draft or proposal document.
- `documents/pp.pdf` — exported presentation or proposal document.

## Notes

This structure is intentionally simple to support future additions such as analysis, drafts, and final publication materials.
